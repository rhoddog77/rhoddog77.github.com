Portfolio

Mavu

Android application developed for my Mobile Application Development class. We were required to work in a team of three to create an app that incorporated many of the Android features we learned throughout the semester. This app is allows users to create and view events in any city they want based on the four categories listed. Users can also view other users from the posts they created.

Shopping Cart

This shopping cart was built for one of my web development classes. We were required to create an interactive shopping cart using PHP as the server side script. The shopping cart required creating an account in order to add an item to your cart and checking out. It also accessed an SQL database to get and modify the data associated with the store and user.

Photo Retouch Project

This photo retouch project was part of the photography class I took. I used Adobe Photoshop to restore the image to near original quality. I was given the opportunity to be an Operation Photo Rescue volunteer because of the quality of work I provided.
